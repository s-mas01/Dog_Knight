# Usage
![DL1](https://user-images.githubusercontent.com/125266372/221795491-47046d01-17a9-494f-bdb0-930642e15994.png)
1. <>CodeのプルダウンよりDownload ZIPを選択しファイルをダウンロードする。  
2. ZIPを解凍する。  
3. ファイル内の"DogKnight.exe"をダブルクリックしてアプリを起動する。
